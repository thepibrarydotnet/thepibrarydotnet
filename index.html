Why Pibrary

Built on Open Source hardware, using Open Source software, Pibrary is a very low cost solution for all your library needs

OPEN SOURCE

Node.js and MongoDB on the server with Bootstrap Angular.js in the browser, Pibrary's responsive interface adapts seemlessly to desktop, tablet and mobile devices.

USER-FRIENDLY

Simple to use user and administrative interfaces make Pibrary quick to learn for librarians and simplicity itself for users of all ages

SELF CONTAINED

Just plug it in to your network and you're ready to go
